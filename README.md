# Шифр перестановки по таблице

Перестановка, усложненная по таблице – это метод шифрования, при котором каждый символ исходного сообщения записывается в таблицу, состоящую из столбцов, зависящих от значения ключа. Поскольку ключ будет состоять из цифр, обозначающих порядок, по которому будет шифроваться сообщение: все символы под цифрами в столбце таблицы – зашифрованный кусок сообщения. Однако в таблице появляются заблокированные ячейки. Заблокированные ячейки встают на место, где должен быть зашифрованный символ, а сам символ становится на следующую позицию. Зашифрованное сообщение получается путем сложения всех зашифрованных кусков. Зашифрованные куски сообщения устанавливаются в порядке по возрастанию цифр ключа.

## Как запустить

1. Открыть table-transposition-cipher.sln в Visual Studio 2022
2. Нажать F5 для компиляции и запуска

## Использование

1. Необходимо выбрать "Шифрование" или "Расшифрование".
2. Необходимо ввести текст.
3. Необходимо ввести ключ.
4. Необходимо указать позицию, где будет блокирование ячейки.
5. Программа демонстрирует таблицу и текст-результат.
